start using git
```
git init
```

check status
```
git status
```

show all difference since last commit
```
git diff
```

git clone without downloading
```
git clone <repo> --no-checkout
```

commit with message
```
git commit -m "<msg>"
```

revert to the last commit
```
git reset HEAD~
```

add repo as submodule
```
git submodule add <repo>
```
